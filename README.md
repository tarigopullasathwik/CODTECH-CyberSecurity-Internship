# CODTECH Cyber Security Internship

This repository contains all four tasks completed as part of the CODTECH Cyber Security Internship.  
Each task demonstrates practical implementation of core cyber security concepts using Python.

---

## Tasks Overview

### Task 1: File Integrity Checker
A Python-based tool that ensures file integrity by generating and comparing cryptographic hash values.  
It helps detect unauthorized changes in files.

Technologies: Python, hashlib

---

### Task 2: Web Application Vulnerability Scanner
A basic vulnerability scanner that identifies common web security issues such as SQL Injection and Cross-Site Scripting (XSS) in web applications.

Technologies: Python, requests, BeautifulSoup

---

### Task 3: Penetration Testing Toolkit
A modular penetration testing toolkit that includes utilities such as a port scanner and brute-force module for basic security testing.

Technologies: Python, socket programming

---

### Task 4: Advanced Encryption Tool (AES-256)
A secure file encryption and decryption tool implemented using AES-256 encryption with password-based authentication.

Technologies: Python, AES-256 (PyCryptodome)
