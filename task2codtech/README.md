# Web Application Vulnerability Scanner

This project is part of the CODTECH Cyber Security Internship (Task 2).

## Objective
To identify common web application vulnerabilities such as:
- SQL Injection
- Cross-Site Scripting (XSS)

## Tools Used
- Python
- requests library

## How It Works
1. Accepts a target URL with parameters
2. Tests SQL Injection using common payloads
3. Tests XSS by injecting script payloads
4. Displays possible vulnerability alerts
