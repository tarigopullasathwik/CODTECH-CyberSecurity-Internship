Description:
This project is a Python-based modular penetration testing toolkit.
It is designed for educational purposes to demonstrate how basic
security testing tools work.

Modules Included:
1. Port Scanner
   - Scans a target system for open TCP ports
   - Uses Python socket programming

2. Brute-Force Module
   - Demonstrates brute-force logic using a password list
   - Implemented as a safe educational simulation

Technologies Used:
- Python
- socket
- paramiko (optional)

How to Run:
1. Install Python
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python main.py
