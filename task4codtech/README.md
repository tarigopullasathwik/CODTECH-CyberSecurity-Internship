## Description
This tool encrypts and decrypts files using AES-256 encryption with password-based key derivation.
It ensures confidentiality and integrity of sensitive files.

## Features
- AES-256 encryption (GCM mode)
- Password-based security
- User-friendly CLI interface
- Secure file handling

## Usage
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python encryption_tool.py

3. Choose encrypt or decrypt option.
